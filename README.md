# Лабораторная работа 2.1

## Описание
Создание и хранение инструментов, записей их калибровок и обслуживаний

**Предметная область:** 3 (Оборудование и калибровки)  
**Вариант коллекции:** HashMap<Long, T>
**Вариант файла:** JSON

---

## Требования
- **Java:** 17+
- **Maven:** 3.3+

---

## Сборка
```bash
# Очистка и сборка
mvn clean package

# Только сборка
mvn package

# Пропустить тесты
mvn package -DskipTests
```
## Запуск
```bash
# Из JAR файла

# Со стандартным файлом data.json
java -jar target/Lab5-1.0-SNAPSHOT.jar

# С указанием пути к файлу данных
java -jar target/Lab5-1.0-SNAPSHOT.jar --data-file=storage/lab_data.json

# Из IDE
Запустите класс ru.itmo.zhmeh.Main
```

## Команды

### Общие команды

| Команда | Описание |
|---------|----------|
| `help` | Список всех команд |
| `exit` | Выход из программы |

### Приборы (Instrument)

| Команда | Описание |
|---------|----------|
| `inst_add` | Добавить прибор (интерактивно) |
| `inst_list [--type TYPE] [--status STATUS]` | Список приборов с фильтрацией |
| `inst_show <id>` | Карточка прибора |
| `inst_update <id> field=value...` | Обновить поле прибора |
| `inst_due [--days N]` | Приборы, требующие калибровки |

### Калибровки (Calibration)

| Команда | Описание |
|---------|----------|
| `cal_add <instrument_id>` | Добавить калибровку (интерактивно) |
| `cal_list <instrument_id> [--last N]` | Список калибровок прибора |
| `cal_show <calibration_id>` | Карточка калибровки |

### Обслуживание (Maintenance)

| Команда | Описание |
|---------|----------|
| `maint_add <instrument_id>` | Добавить обслуживание (интерактивно) |
| `maint_list <instrument_id> [--last N]` | Список обслуживаний прибора |
